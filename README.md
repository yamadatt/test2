# test2

## How to run the Golang application

1. Make sure you have Golang installed on your system. You can download it from [https://golang.org/dl/](https://golang.org/dl/).
2. Clone the repository:
   ```
   git clone https://github.com/yamadatt/test2.git
   ```
3. Navigate to the repository directory:
   ```
   cd test2
   ```
4. Run the application with two arguments:
   ```
   go run main.go <arg1> <arg2>
   ```
   Replace `<arg1>` and `<arg2>` with the two numbers you want to add.

## How to run the tests

2. Clone the repository:


## GitHub Actions Workflow for Running Tests on Push Events

This repository includes a GitHub Actions workflow that runs tests automatically whenever a push event occurs. The workflow file is located in the `.github/workflows` directory.

To view or modify the workflow, navigate to the `.github/workflows/test.yml` file.
