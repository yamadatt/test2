# test2
イノベージョン創出会議

## Usage

To run the application, use the following command:

```
go run main.go <num1> <num2>
```

Replace `<num1>` and `<num2>` with the two numbers you want to add.

## Example

```
$ go run main.go 3 5
8
```

## Testing

To run the tests, use the following command:

```
go test
```
