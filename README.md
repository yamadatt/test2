## Usage

To run the Golang application and display the sum of two arguments, use the following command:

```sh
go run main.go <arg1> <arg2>
```

Replace `<arg1>` and `<arg2>` with the two numbers you want to add.

## Running Unit Tests

To run the unit tests for the Golang application, use the following command:

```sh
go test
```

## Continuous Integration

This project uses GitHub Actions for continuous integration. Unit tests are automatically executed on every push to the repository.

To view the CI results, go to the "Actions" tab of the repository on GitHub.
