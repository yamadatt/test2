## Usage

To run the Golang application and display the sum of two arguments, use the following command:

```sh
go run main.go <arg1> <arg2>
```

Replace `<arg1>` and `<arg2>` with the two numbers you want to add.

## Running Unit Tests

To run the unit tests for the Golang application, use the following command:

```sh
go test
```
